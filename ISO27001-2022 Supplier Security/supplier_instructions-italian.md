# Istruzioni per l’Autosvalutazione del Fornitore (Italiano)

Grazie per completare la nostra autosvalutazione di sicurezza informatica basata su ISO 27001:2022. Il questionario HTML funziona completamente offline nel browser e può essere salvato in qualsiasi momento. Seguite i passaggi riportati di seguito per descrivere accuratamente il vostro stato di sicurezza.

## 1. Prima di Iniziare
- Utilizzate un browser desktop aggiornato (Chrome o Edge consigliati; Firefox e Safari sono supportati).
- Scaricate sul vostro computer il file `supplier-cybersecurity-assessment.html`.
- Verificate che il browser consenta i download di file e l’esecuzione di JavaScript. Potrebbe essere necessario autorizzare temporaneamente i pop-up generati dal modulo di salvataggio.

## 2. Aprite il Questionario
- Individuate il file HTML scaricato e fate doppio clic per aprirlo nel browser.
- L’autovalutazione viene eseguita localmente; non è richiesta la connessione a Internet e nessun dato viene inviato ai nostri sistemi finché non ci restituite il file salvato.
- Controllate che la data della valutazione nella parte superiore corrisponda a oggi. Questa data viene salvata con le vostre risposte ed è inclusa nel nome del file restituito.

## 3. Compilate le Informazioni Aziendali
- Completate tutti i campi obbligatori nella sezione “Informazioni sul Fornitore”:
  - Nome dell’azienda
  - Referente principale
  - Indirizzo email
  - Numero di telefono
  - Data della valutazione (precompilata con la data odierna; modificatela se necessario)
- È necessario compilare questi campi per poter salvare l’autovalutazione.

## 4. Rispondete alle 39 Domande di Sicurezza
- Il questionario copre 10 domini di controllo in linea con ISO 27001:2022.
- Per ogni domanda scegliete il livello di maturità che descrive meglio l’implementazione attuale:
-  Le opzioni descrivono livelli crescenti di maturità: da `Non Implementato` a `Completamente Maturo`; `Non Applicabile` è disponibile quando il controllo non è pertinente.
- Fornite risposte sincere; potete aggiungere chiarimenti nel campo “Commenti aggiuntivi”.
- Potete lasciare alcune domande in bianco e tornare più tardi. La barra di avanzamento e il punteggio di maturità si aggiornano automaticamente mentre selezionate le risposte.

## 5. Salvate le Risposte
- Quando siete pronti scorrete in fondo alla pagina e fate clic su **“Salva la valutazione sul disco.”**
- Il browser scarica un nuovo file HTML denominato `security_assessment-<vostroazienda>-<YYYY-MM-DD>.html`.
- Il file salvato contiene tutte le risposte inserite fino a quel momento. Potete salvare più volte; ogni salvataggio aggiorna i dati incorporati e la data nel nome del file.

## 6. Riprendete in un Secondo Momento (Opzionale)
- Per continuare successivamente aprite l’ultimo file `security_assessment-...html` salvato.
- Tutte le informazioni inserite, il punteggio e gli indicatori di avanzamento appariranno esattamente come li avete salvati.
- Aggiornate le risposte se necessario e fate nuovamente clic su **“Salva la valutazione sul disco”** quando avete terminato.

## 7. Inviate la Valutazione Completata
- Entro la scadenza richiesta inviate via email l’ultimo file HTML salvato al referente indicato.
- Non inviate screenshot o PDF; abbiamo bisogno del file HTML per mantenere i dati facilmente elaborabili.
- Se disponete di evidenze o piani di rimedio, allegateli separatamente o indicate come possiamo consultarli.

## 8. Serve Assistenza?
- Per chiarimenti sul questionario o sulle nostre aspettative di sicurezza contattate la persona che ha richiesto la valutazione.
- In caso di problemi tecnici (ad esempio il download non parte) provate un browser supportato diverso o consentite temporaneamente i pop-up/ download per i file locali.
- Se dopo l’invio individuate informazioni errate, aggiornate il questionario, salvatelo di nuovo e reinviatelo indicando nell’oggetto che si tratta di una versione aggiornata.

Grazie per la collaborazione. Le vostre risposte ci aiutano a valutare e migliorare la sicurezza dell’intera catena di fornitura.
