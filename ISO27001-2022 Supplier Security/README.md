# ISO 27001:2022 Supplier Cybersecurity Self-Assessment

A comprehensive, self-contained HTML assessment tool for suppliers to evaluate and document their cybersecurity maturity level in alignment with ISO 27001:2022 standards.

## Overview

This tool enables suppliers to complete a detailed cybersecurity self-assessment covering 10 key security domains with 39 questions. The assessment saves locally as an HTML file that can be reopened, modified, and shared via email.

## Key Features

- **Self-contained HTML file** - No server, database, or internet connection required
- **Automatic progress tracking** - Visual progress bar updates as you answer questions
- **Real-time maturity scoring** - Instant score calculation based on your responses
- **Answer persistence** - Saves answers directly into the HTML file when downloaded
- **Reopenable & editable** - Can reopen and modify answers before final submission
- **Multi-language support** - Available in English, Spanish, French, and German

## Quick Start Guide for Suppliers

### Step 1: Open the Assessment
1. Download the `supplier-cybersecurity-assessment.html` file
2. Double-click the file to open it in your web browser
3. The assessment will open offline - no internet connection needed

### Step 2: Complete Company Information
Fill in the required fields at the top:
- Company Name
- Contact Person
- Email Address
- Phone Number
- Assessment Date (auto-populated with today's date)

### Step 3: Answer Security Questions
The assessment covers 10 security domains:
1. Information Security Policy & Governance
2. Asset Management
3. Access Control
4. Data Protection
5. Physical & Environmental Security
6. Operations Security
7. Network Security
8. Incident Management
9. Business Continuity
10. Compliance & Monitoring

For each question, select your maturity level:

| Level | Points | Description |
|-------|--------|-------------|
| **Not Implemented** | 0 | Control not in place |
| **Partially Implemented** | 1 | Control partially in place, inconsistent application |
| **Implemented** | 3 | Control in place and consistently applied |
| **Fully Mature** | 5 | Control fully integrated, monitored, and continuously improved |
| **Not Applicable** | N/A | Question doesn't apply to your organization |

### Step 4: Track Your Progress
- Progress bar shows completion percentage (based on answered questions)
- Score updates automatically as you answer questions
- View your current maturity score in real-time

1. Click the **"ðŸ’¾ Save Assessment to Disk"** button at the bottom
2. Your browser will download a file named: `security_assessment-[yourcompany]-[date].html`
3. The file contains all your answers and can be reopened later

### Step 6: Review or Modify (Optional)
1. Locate the saved file in your Downloads folder
2. Double-click to reopen it in your browser
3. All your previous answers will be displayed
4. Make any changes needed
5. Click **"ðŸ’¾ Save Assessment to Disk"** again to save the updated version
6. Note: The date in the filename will update to reflect when you last saved it

1. Email the final saved HTML file to your client contact
2. They can open it in any browser to review your responses
3. Your maturity score will be visible in the document

## Scoring System

### Score Calculation
- **Maximum possible score:** 195 points (39 questions Ã— 5 points each)
- **Scoring is cumulative:** Your total score is the sum of all answered questions
- **"Not Applicable" answers** do not contribute to the score

### Maturity Levels

| Score Range | Maturity Level | Description |
|-------------|----------------|-------------|
| **0** | No Assessment | Assessment not started |
| **1-50** | Initial | Significant improvements needed |
| **51-100** | Developing | Basic security practices in place |
| **101-150** | Defined | Good security practices established |
| **151-195** | Optimized | Mature security practices |
